🔍 How Greedy Algorithm Works
Scoring Formula:
recommendation_score = (100 - progress) + priority_weight
Priority Weights:
  • 🔴 High: +30 points
  • 🟡 Medium: +20 points
  • 🟢 Low: +10 points
Algorithm Strategy:
  1. Calculate: Compute score for each skill
  2. Select: Choose highest scoring skill
  3. Repeat: Continue until all ranked
  4. Recommend: Suggest top N skills
Why Greedy?

Greedy approach selects the locally optimal choice (highest score) at each step, which typically gives good results for skill prioritization without needing complex optimization.
